# 📝 To-Do List Application (Python CLI)

A simple command-line To-Do List application built using Python.
This project demonstrates list manipulation, loops, conditional logic, and menu-driven programming.

---

## 📌 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* 🚪 Exit program safely

---

## 🛠 Technologies Used

* Python 3
* Lists (Core DSA Concept)
* Loops
* Conditional Statements
* Command Line Interface (CLI)

---

## 📂 Project Structure

```
todo.py
README.md
```

## 🚀 How to Run

1. Clone the repository:

```
git clone <your-repository-link>
```

2. Navigate into the project folder:

```
cd todo-list-python
```

3. Run the program:

```
python todo.py
```
